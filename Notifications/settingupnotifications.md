# Setting Up Notifications

You can configure the Redwood Cloud Portal so that you will receive notifications when problems occur with your RunMyJobs SaaS instance.

1. In the Redwood Cloud Portal, navigate to *Environments > Settings* in the Navigation Bar.
2. Under *Notifications*, check the desired notification types. The options are:

    - *Environment Restarted*: A notification is sent if the RunMyJobs instance restarts unexpectedly.
    - *Secure Gateway Down*: A notification is sent if the RunMyJobs Secure Gateway terminates unexpectedly.
    - *Secure Gateway Is Up Again*: A notification is sent if the RunMyJobs Secure Gateway restarts after terminating unexpectedly.
