# Help

The Help section of the navigation bar includes links to the following sites:

- *Portal Documentation*: This site.
- *Product Documentation*: The documentation site for the selected Redwood product. Where is this selected?
- *Product Support*: The Redwood support site, where you can look up Knowledge Base entries and submit support cases.
- *Redwood Cloud Status*: A site where you can monitor the health of Redwood Cloud Services.
- *Redwood University*: The main training site for Redwood products. Does this always take the customer to the main Redwood RU site, or are there customer-specific RU sites?
